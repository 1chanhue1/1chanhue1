## 1. 프로젝트 소개
  
**현대인이라면 누구나 도파민 중독?**
**코스믹 디톡스와 함께 우주여행 하며 디지털 디톡스 해보세요!**

**휴대폰을 놓지 못하는 현대인들, 이제는 디지털도 다이어트를 해보는 건 어떨까요?**

**쉽지 않은 도전을 코스믹 디톡스가 도와줄게요.**
